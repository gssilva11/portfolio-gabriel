# Portfolio — Gabriel Santos

Portfólio profissional (Next.js + TypeScript) com foco em experiência do usuário, animações suaves e páginas de projeto por slug.

**Live:** https://portfolio-gabriel-zeta.vercel.app/

## Stack
- Next.js (App Router)
- React + TypeScript
- TailwindCSS
- Framer Motion
- Resend (contato por e-mail)
- Vercel (deploy)

## Funcionalidades
- Landing page com seções (Hero, Projetos, Skills, Sobre e Contato)
- Lista de projetos e páginas individuais (`/projetos/[slug]`)
- Navegação entre projetos (anterior/próximo)
- Formulário de contato com validação e antispam simples
- SEO básico + OpenGraph

## Rodar localmente
```bash
npm install
npm run dev
